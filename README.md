# Caribbean Quest

An ADHD-friendly Grade Six Social Studies quiz game built from the themes in the CV 2017-2025 Word files.

## Open

Double-click `index.html` to play in a browser.

## Features

- Grade Six Social Studies question bank based on the CV files
- Caribbean family, health, rights, culture, history, geography, resources, and citizenship topics
- Exact question images extracted from the CV Word documents
- A year is selected before the game starts
- Questions run in order from the selected year's paper, up to questions 1-50
- Years with fewer than 50 extracted questions are shown but disabled in the picker
- Exact images display for questions where an image was extracted from the source document
- Responsive layout for mobile phones, tablets, and desktop screens
- Large answer buttons and short 10-question rounds
- Instant correct or incorrect feedback after each answer
- Correct answers award coins and incorrect answers deduct coins
- Live coin score throughout the game
- End review for each incorrect answer with the question, player answer, correct answer, and review tip
- Hint button for low-pressure practice
- Sound effects made with the browser Web Audio API
- Calm mode to turn off animation
- Keyboard controls: `1`, `2`, `3` to answer, `H` for hint, `Enter` for next
